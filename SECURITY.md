# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 2.0.x   | Yes                |
| < 2.0   | No                 |

## Reporting a Vulnerability

If you discover a security vulnerability in Aspectly, please report it responsibly.

**Do not open a public issue.**

Instead, email **jeanisahakyan@gmail.com** with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You will receive a response within 48 hours. We will work with you to understand and address the issue before any public disclosure.

## Scope

This policy covers:

- `@aspectly/core` (npm)
- `@aspectly/transports` (npm)
- `@aspectly/react-native` (npm)
- `@aspectly/react-native-web` (npm)
- `@aspectly/web` (npm)
- `Aspectly.Bridge` (NuGet)
- `Aspectly.Bridge.CefSharp` (NuGet)
- `Aspectly.Bridge.WebView2` (NuGet)
