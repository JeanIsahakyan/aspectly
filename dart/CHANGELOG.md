# Changelog

## 2.1.0

- Initial release of `aspectly_bridge` for Dart / Flutter.
- `BridgeHost` (pure Dart) speaking the Aspectly protocol, plus the
  `BrowserBridge` abstraction for any web view (e.g. `webview_flutter`).
- Auto-detected on the web side via the `@aspectly/transports` Flutter transport.
